# ¿Qué es el montaje de un sistema de archivos?

## Definición

El montaje de un sistema de archivos es el proceso mediante el cual el sistema operativo hace accesible una partición o dispositivo de almacenamiento para que el usuario pueda leer y escribir información en él.

---

## ¿Para qué sirve?

El montaje permite:

- Acceder a los archivos almacenados en una partición.
- Leer y escribir información en un dispositivo.
- Integrar el dispositivo al árbol de directorios del sistema operativo.

---

## Importancia en este proyecto

Después de crear la nueva partición y el sistema de archivos NTFS, fue necesario montar el disco para comprobar que podía utilizarse correctamente.

La validación confirmó que Linux Mint montó la partición sin errores y permitió crear, modificar y eliminar archivos.

---

## Lo que aprendimos

Que un disco puede ser reconocido por el sistema operativo sin estar montado. Solo después del montaje es posible acceder a su contenido y verificar que funciona correctamente.
