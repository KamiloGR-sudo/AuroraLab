# Reparación de Disco Externo ADATA HD650

## Descripción

Este repositorio documenta el proceso completo de diagnóstico, reparación y validación de un disco duro externo ADATA HD650 de 1 TB que presentó una corrupción del sistema de archivos NTFS.

El objetivo del proyecto fue identificar la causa real de la falla, aplicar una metodología de diagnóstico segura y documentar cada paso para que pueda servir como guía en futuros casos similares.

## Resultado

- Hardware validado mediante SMART.
- Diagnóstico de corrupción lógica del sistema de archivos NTFS.
- Reconstrucción de la partición.
- Recreación del sistema de archivos NTFS.
- Validación completa en Linux Mint.
- Validación completa en Windows 10.
