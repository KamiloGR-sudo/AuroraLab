# AuroraLab

> Comprender antes que modificar.

AuroraLab es mi laboratorio personal de investigación, aprendizaje e ingeniería.

Este proyecto nació con el objetivo de comprender cómo funcionan los sistemas antes de intentar modificarlos. Cada experimento, descubrimiento y error quedará documentado para que el conocimiento no se pierda y pueda servir tanto para mi aprendizaje como para otras personas.

## Objetivos

- Aprender Linux desde sus fundamentos.
- Dominar Git y GitHub.
- Documentar investigaciones técnicas.
- Comprender firmware, Bluetooth y hardware.
- Desarrollar proyectos de ingeniería con una metodología basada en la experimentación.

## Filosofía

No memorizar comandos.

Comprender sistemas.

No buscar respuestas rápidas.

Construir conocimiento.
