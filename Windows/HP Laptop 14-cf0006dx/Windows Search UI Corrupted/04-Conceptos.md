# Conceptos

## Windows Search

Componente de Windows encargado de indexar archivos, aplicaciones y configuraciones para permitir búsquedas rápidas desde el menú Inicio y la barra de búsqueda.

---

## SFC (System File Checker)

Herramienta de Windows que verifica la integridad de los archivos protegidos del sistema y reemplaza automáticamente aquellos que estén dañados o modificados.

---

## DISM (Deployment Image Servicing and Management)

Herramienta utilizada para comprobar y reparar la imagen de Windows. Puede corregir problemas que SFC no puede solucionar por sí solo.

---

## Windows PowerShell

Consola de administración avanzada de Windows que permite ejecutar comandos y automatizar tareas administrativas.

---

## Registro de aplicaciones (AppX)

Proceso mediante el cual Windows vuelve a registrar una aplicación del sistema utilizando su archivo `AppXManifest.xml`, restaurando su configuración cuando presenta fallos.
